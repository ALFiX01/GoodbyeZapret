## 📌 О проекте

**GoodbyeZapret** — это инструмент на базе [zapret-win-bundle/zapret-winws](https://github.com/bol-van/zapret-win-bundle/tree/master/zapret-winws), предназначенный для модификации сетевых пакетов с целью предотвращения принудительного разрыва сессий системами DPI (Deep Packet Inspection).

*Инструмент изменяет сетевые пакеты так, чтобы DPI-системы не могли корректно определить ресурс.*

## ⚠️ Важное предупреждение об антивирусах

> [!CAUTION]
> **WinDivert и Launcher могут определяться как `HackTool`, `RiskTool` или `trojan`**, так как:
> 
> 1. `WinDivert` перехватывает и модифицирует сетевые пакеты
> 2. `Launcher` вносит изменения в реестр для автозапуска и настройки UAC

**Что делать:**
- Добавьте папку проекта в исключения антивируса
- Не используйте проект, если не доверяете коду

## 📦 Установка

### Метод 1. PowerShell (Самый простой)

*Перед установкой временно отключите антивирус — это стандартная рекомендация для корректной работы установщика*

Откройте PowerShell **от имени администратора** и выполните команду:

```powershell
irm https://raw.githubusercontent.com/ALFiX01/GoodbyeZapret/refs/heads/main/Files/script/install.ps1 | iex
```

Если выполнение скриптов запрещено политикой, используйте:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/ALFiX01/GoodbyeZapret/refs/heads/main/Files/script/install.ps1 | iex"
```

---

### Метод 2. installer (автоматическая установка)

1. Скачайте файл [`installer.exe`](https://github.com/ALFiX01/GoodbyeZapret/raw/refs/heads/main/Files/Updater/installer.exe) и переместите его в любую удобную папку (например, `C:\`).
2. Запустите `installer.exe` **от имени администратора** и следуйте подсказкам в открывшемся окне.
3. После завершения рядом с `installer.exe` появится папка `GoodbyeZapret` — в неё будет установлена программа.
4. Для запуска перейдите в созданную папку и откройте `Launcher.bat` **от имени администратора**.

---

### Метод 3. Архив (ручная установка)

1. Скачайте архив [`GoodbyeZapret.zip`](https://github.com/ALFiX01/GoodbyeZapret/raw/refs/heads/main/Files/GoodbyeZapret.zip).
2. Распакуйте архив в удобное место, например, `C:\GoodbyeZapret`  
   > ⚠️ Путь к папке GoodbyeZapret не должен содержать пробелов, специальных или кириллических символов.
3. Проверьте, что путь к исполняемому файлу похож на:  
   ```
   <Системный диск>:\GoodbyeZapret\Launcher.bat
   ```
4. Запустите `Launcher.bat` **от имени администратора**.

## 🔄 Обновление

### Автоматически

Запустите:

```
<Системный диск>:\GoodbyeZapret\tools\Updater.exe
```
> ⚠️ Тоже запускать от имени администратора.

### Вручную

1. Перейдите на страницу [последнего релиза](https://github.com/ALFiX01/GoodbyeZapret/releases/latest) и скачайте архив `GoodbyeZapret.zip`
2. Распакуйте скачанный архив `GoodbyeZapret.zip`
3. Завершите работу конфига и 
GoodbyeZapretTray.exe, а также закройте консоль, связанную с GoodbyeZapret. Это необходимо для корректного обновления.
4. Удалите **всё содержимое** папки с установленной программой, но не саму папку. По умолчанию путь:
   ```
   <Системный диск>:\GoodbyeZapret
   ```
*(Если вы устанавливали в другую папку — очистите её содержимое.)*

5. Переместите содержимое распакованного архива в эту же папку *(пункт 4)*.
6. Запустите `Launcher.bat` **от имени администратора**.

## ▶️ Быстрый старт

1. Откройте папку:
   ```
   <Системный диск>:\GoodbyeZapret\configs
   ```
2. Запустите нужный `.bat` файл **от имени администратора**:
   - `MultiFix.bat` — для обхода блокировок Discord, YouTube и др.
   - `MultiFix_AllPort.bat` — аналог, но с полным диапазоном портов.
   - `MultiFix_Lab.bat` — детально настраиваемый через блокнот.
   - `smart-config.bat` — умный конфиг, который сам подбирает стратегии к заблокированным ресурсам в режиме реального времени (может работать не стабильно).
   - Альтернативные варианты: `MultiFix_2.bat`, `MultiFix_3.bat` и др.
3. **Оставьте консоль открытой** — она должна работать для обхода блокировок.
4. Для остановки — просто закройте окно консоли.

## 🔁 Автозапуск

1. Запустите файл:
   ```
   C:\GoodbyeZapret\Launcher.bat
   ```
2. В открывшемся меню:
   - Выберите нужный скрипт, указав его номер.
   - Подтвердите установку в автозагрузку.
3. Чтобы удалить автозапуск, снова запустите:
   ```
   C:\GoodbyeZapret\Launcher.bat
   ```
   - выберите пропишите `DS` (удаление из автозагрузки).

## 🛠️ Устранение неполадок

### 🚫 Нет эффекта

- Убедитесь, что `.bat`-файл запущен **от имени администратора**.
- Проверьте корректность пути к скрипту:
  ```
  <Системный диск>:\GoodbyeZapret\configs\...
  ```
- Попробуйте использовать другой конфиг.

### 🧹 Файлы удаляются или проблема с запуском Launcher.bat или конфигов

- Добавьте папку в исключения антивируса:
  ```
  <Системный диск>:\GoodbyeZapret
  ```
- Переустановите GoodbyeZapret, следуя [инструкции по установке](#-установка).

### ⚠️ Нестабильная работа

- Попробуйте другой конфиг.
- Отключите VPN, если они используются.
- Проверьте параметры в браузере:
  - Перейдите по адресу `chrome://flags`
  - Убедитесь, что **QUIC** и **TLS 1.3** установлены в значение **Default**.

### ❗ Ошибки при запуске

- Убедитесь, что путь установки **не содержит пробелов, спец. символов и кириллицы**.
- GoodbyeZapret должен быть установлен на **системном диске**, например:
  ```
  <Системный диск>:\GoodbyeZapret
  ```

---

### 🧪 Быстрая диагностика

`Launcher.bat` может автоматически выполнить проверку системы на наличие потенциальных проблем которые могут повлиять на работу программы.

1. Откройте:
   ```
   <Системный диск>:\GoodbyeZapret\Launcher.bat
   ```
2. В меню выберите опцию:
   ```
   Состояние GoodbyeZapret
   ```
   Если проблемы будут обнаружены, программа сообщит вам о них в открывшемся меню.
